Wrapping Other Thenables

Angular's $q can also be used to wrap promises from other sources.

- $q.when() takes a promise and returns an Angular promise. - Integrates into Angular's model observation.

$q.when() can also be useful when writing tests and mocks.

- $q.when() can also take a value and return a promise that resolves immediately. - Good for mocking methods which return promises. - Prevents complicated shims and code jank - client code doesn't change.
